# 华南师范大学「晚安华师」微信公众号日程自动抓取工具

## 项目概述

自动抓取微信公众号「晚安华师」推文，并提取推文中的日程信息的 Python 工具。

**v6.0 全新版本**：100% 全自动，零交互，双击即可运行！

## 功能特性

### v6.0 新功能（全自动版本）

1. **零交互运行** - 无需任何用户输入，程序启动后全自动执行
2. **自动访问新闻网** - 自动访问华南师范大学新闻网 (https://news.scnu.edu.cn/)
3. **智能链接筛选** - 自动检测 302 跳转，只保留跳转到 mp.weixin.qq.com 的链接
4. **自动批量抓取** - 自动抓取所有有效微信文章
5. **智能数据校验** - 来源必须是「晚安华师」，标题、正文长度自动校验
6. **按时间排序** - 以微信官方发布时间为准，取最新 2 篇
7. **日程自动提取** - 智能识别时间、活动、会议、地点等日程内容
8. **多格式输出** - 控制台打印 + markdown 文件自动保存

### v5.x 功能（手动版本）

- 手动输入文章链接
- 自动提取正文和日程信息
- 数据校验 + 兜底 API 验证

## 运行方法

### 全自动版本（推荐）

```bash
python src/auto_scnu_crawler.py
```

程序启动后自动执行：
1. 访问华南师范大学新闻网
2. 提取所有文章链接
3. 检测 302 跳转，筛选微信链接

**全程无需任何用户输入！**

```bash
python src/scnu_wechat_crawler.py
```

程序启动后：
1. 输入微信公众号文章链接（mp.weixin.qq.com/s/xxx）
2. 按回车开始处理
3. 可继续输入更多链接
4. 输入 `q` 退出

## 可配置参数

在 `auto_scnu_crawler.py` 文件开头可配置以下参数：

```python
# 目标网站
NEWS_SCNU_EDU_URL = "https://news.scnu.edu.cn/"

# 兜底 API 地址
FETCHER_API_URL = "https://down.mptext.top/api/public/v1/download"

# 正文最小长度阈值
MIN_CONTENT_LENGTH = 500

# 请求间隔（秒）
FETCH_DELAY = 2

# API 超时时间（秒）
API_TIMEOUT = 15

# 需要获取的最新文章数量
TARGET_ARTICLE_COUNT = 2  # 修改此值以获取更多文章

# 输出文件路径
OUTPUT_MARKDOWN_FILE = "wechat_schedule_output.md"
```

## 注意事项

1. **新发布文章同步延迟** - 新发布的文章需等待 10-30 分钟才能被抓取
2. **微信安全策略** - 程序自动使用微信 UA（MicroMessenger）绕过安全检测
3. **请求间隔控制** - 默认 2 秒间隔，避免触发反爬
4. **仅处理微信链接** - 只处理跳转到 mp.weixin.qq.com 的链接，其他链接自动忽略

## 规避的反爬措施

1. ✅ 使用微信 UA（MicroMessenger/7.0.20.1781）
2. ✅ 不直接裸请求 mp.weixin.qq.com
3. ✅ 请求间隔控制（FETCH_DELAY = 2 秒）
4. ✅ 自动检测 302 跳转
5. ✅ 不使用搜狗微信搜索（避免反爬、数据陈旧）

## 常见问题

### Q: 程序运行后没有抓取到任何文章？

A: 可能原因：
1. 网络连接问题，无法访问华南师范大学新闻网
2. 当前新闻网没有「晚安华师」的文章
3. 所有文章链接都不跳转到微信公众号

解决方法：
- 检查网络连接
- 手动访问 https://news.scnu.edu.cn/ 查看是否有相关文章
- 稍后重新运行程序

### Q: 如何获取更多文章？

A: 修改 `auto_scnu_crawler.py` 中的 `TARGET_ARTICLE_COUNT` 参数：

```python
TARGET_ARTICLE_COUNT = 5  # 获取最新 5 篇
```

### Q: 抓取的文章来源不是「晚安华师」？

A: 程序会自动过滤非「晚安华师」来源的文章。如需抓取其他来源，修改 `validate_article()` 函数中的来源名称参数。
